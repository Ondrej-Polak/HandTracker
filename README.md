# ✋ Real-Time Hand Tracking with MediaPipe

A lightweight Python application that uses Computer Vision to track hand landmarks in real-time. This project utilizes the stable legacy version of MediaPipe for high-performance skeletal mapping.

## 🚀 Features
- **Skeletal Overlay:** Maps 21 distinct hand landmarks (joints and fingertips).
- **Multi-Hand Support:** Detects and tracks up to 2 hands simultaneously.
- **Dynamic Styles:** Uses `drawing_styles` for professional, color-coded visual feedback.
- **Mirror View:** Automatically flips the video feed for a natural user experience.
